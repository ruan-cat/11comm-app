# 一次性完成大多数页面的批量迁移

1. 阅读 `.github\prompts\route-migration-map.yml` 文件，了解清楚旧项目和本项目的路由映射关系。
2. 阅读 `docs\reports\2026-01-13-index-work-entry-analysis.md` 和 `docs\reports\2026-01-13-vue2-to-vue3-home-workbench-migration-analysis.md` 报告文档，了解清楚目前首页和工作台缺少那些应用访问入口。
3. 阅读 `docs\reports\vue2-route-navigation-map.md` 报告，了解清楚旧代码之间的路由跳转关系。
4. 全面阅读 `.claude\skills` 内的全部技能文件，了解清楚迁移代码和具体编写代码时的编写规范。
5. 根据 `index-work-entry-analysis` 和 `vue2-to-vue3-home-workbench-migration-analysis` 报告的内容，设计一份非常完整，完善，齐全的迁移任务清单。并根据 `.claude\skills` 内的全部技能文件的要求，设计清晰明确的 spec 操作规范。
